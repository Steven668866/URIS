# Archive 目录说明

本目录用于存放当前平台主线（`app.py` + `src/uris_platform/`）之外的历史资料、旧脚本和大体积归档文件，目的是保持根目录结构清晰。

当前归档分组：

- `legacy-video-reasoning-root/`：旧版单文件 Video Reasoning Assistant 的根目录文档、脚本、Colab 文件、旧测试样本
- `legacy-homerobot-core/`：历史家居机器人相关子项目代码（当前平台未直接依赖）
- `model-artifacts/`：大体积压缩包/模型归档文件（例如历史 zip 包）

说明：

- 归档不代表删除，只是为了整理结构。
- 当前平台运行主线仍在根目录和 `src/uris_platform/`。
